A code example for illustration of how to process the recorded Moire patterns to retrieve angles and periods. Then, the algorithm illustrates how to fit the model to accurately determine geometrical parameters of dual-phase grating interferometry. data_20230809.pickle is one example used for plotting. The raw images are not uploaded here.

align_tools.py is used as a package for managing corresponding functions.
alignment.py is the processing codes.
data_20230809.pickle is one example of data for plotting.
